Run the Encryption.sh script in command line with the value to encrypt: 
	"$ bash Encryption.sh <VALUE TO ENCRYPT>" 
	
It will encrypt the value using aes-128-cbc

The inputted value will be stored in input.txt. The encrypted value will be stored in enc.txt.

Run the Decryption.sh script in command line:
	"$ bash Decryption.sh"
	
It will decrypt whatever value is stored inside enc.txt, and store the decrypted value in out.txt.

The password used to encrypt/decrpt the values is: ABCDEF
